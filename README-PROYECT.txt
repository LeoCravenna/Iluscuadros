ILUSCUADROS es un e-commerce para adquirir cuadros con diseños ilustrados de distintos personajes dentro de diversas categorías como ser la música, el deporte, películas, etc. 
El sitio web, está creado con React, es una SPA (single page application), podrás navegar por las diferentes rutas
y componentes de forma más optimizada. Actualmente se encuentra en desarrollo.

Utilicé la librería Firebase para obtener la colección de productos que alimenta mi catálogo y para poder crear una colección de Ordenes y una colección de Consultas.

Utilicé la librería Formik para manejar los datos del Formulario tanto para el componente "ContactMe" como para el componente "OrderForm".

Utilicé la librería Yup para crear validaciones dentro de Formik en el Formulario tanto para el componente "ContactMe" como para el componente "OrderForm".

Utilicé la librería SweetAlert2 para mejorar la visual de las alertas al enviar Consultas como también, para cuando se finaliza el pedido y muestra en número de Orden.

Utilicé la librería React Bootstrap para generar el "Badge" en el icono del carrito de compras, que muestra la cantidad de unidades total.

Utilicé la librería Semantic-ui para brindarle los estilos generales al e-commerce en diversos componentes.

Utilicé la librería React-Router-Dom para poder hacer ruteos de un componente a otro.
